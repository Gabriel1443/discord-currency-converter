# Welcome to Currency-Converter

This is a discord bot that converts currency between USD and RMB.

## To have things working alone in this repo
	
	```console
	$ git clone git@github.com:Gabriel1443/trademe-search-engine.git
	$ cd trademe-search-engine
	$ python3 -m venv env
	$ source env/bin/activate
	$ pip install pip-tools
	(env) $ pip-sync
	```

    or just `pip install -r requirements.txt`.  
	
## comments in discord

    - `!rmb 100`: convert $100 RMB to USD.
    sample result: `RMB 100.0 is equivalent to 15.3681 in USD`

    - `!usd 100`: convert $100 USD to RMB.
    sample result: `USD 100.0 is equivalent to 650.7 in RMB`
